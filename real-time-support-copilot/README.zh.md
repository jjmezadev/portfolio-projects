# 实时支持 Copilot

该项目面向客户支持运营，通过整合客户上下文、语义检索、回复建议和下一步动作推荐，帮助人工客服在实时会话中更快处理问题。

## 问题

客服人员经常需要在 CRM、知识库、历史工单和内部政策之间来回切换，导致处理时长上升、回答不一致，并最终影响客户满意度。

## 价值主张

- 基于统一客户上下文生成回复建议。
- 根据意图提供操作建议和下一步动作。
- 支持聊天、邮件和语音等多渠道。
- 具备质量控制、合规和 human-in-the-loop 治理。

## 高层架构

- 对接 CRM、工单、计费和知识系统。
- RAG 层检索文章、历史工单和政策文档。
- Prompt 编排层提供语气控制和 guardrails。
- 面向客服的侧边 UI 提供摘要、行动项和辅助起草。

## 关键交付物

- 统一客户时间线。
- 实时建议引擎。
- 回复质量评分与转人工指导。
- 准确性、实用性和节省时间的离线评估。

## 路线图

### 调研阶段

- 确定优先渠道、意图类型与源系统。

### 原型阶段

- 构建检索、上下文摘要和回复草稿能力。

### 实施阶段

- 接入客服工作台、遥测和策略控制。

### 交付阶段

- 提供采用指南、版本化 prompts 和运营看板。

# 可视化 ETL Builder AI

该平台用于通过拖拽方式设计 ETL 或 ELT 流水线，并提供 AI 辅助 SQL 或转换逻辑生成、部署和运行监控能力。

## 问题

许多数据团队仍然依赖分散的 notebook、脆弱的 SQL 脚本和缺乏治理的临时流水线。构建和维护复杂转换需要较高技术门槛，拖慢数据交付速度。

## 价值主张

- 面向技术与半技术用户的可视化流水线设计。
- AI 辅助生成 SQL 与转换逻辑。
- 集成版本管理、测试和可观测性。
- 可发布到 dbt、Spark 或云数仓目标环境。

## 高层架构

- 具备 source、join、filter 和 destination 节点的 DAG 编辑器。
- 将自然语言转换为转换逻辑的 AI Copilot。
- 编译到 SQL、Python 或编排清单的生成器。
- 运行监控、数据血缘和质量视图。

## 关键交付物

- 可视化编辑器与流水线元模型。
- 带验证能力的可靠 SQL 生成器。
- 初始数仓和调度集成。
- 基于真实数据场景的端到端演示。
